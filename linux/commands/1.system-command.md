### 1. 시스템 명령어
- 리눅스 시스템을 관리하기 위한 명령어

| 명령어 | 특징 |
| :----: | ------------------------------------------- |
| uname | 시스템 정보 표시 |
| crontab | 주기적으로 실행하고 싶은 명령어 등록 |
| free | 메모리 상태 확인 |
| top | 프로세스 정보를 실시간으로 표시 |
| ps | 프로세스 정보 표시 |
| kill | 프로세스 중지 |

- uname
<!-- uname소개<br>-->
```console
foo@bar:~$ uname
foo
```
- crontab
```console
foo@bar:~$ uname
foo
```
- free
```console
foo@bar:~$ uname
foo
```

- top
```console
foo@bar:~$ uname
foo
```

- ps
```console
foo@bar:~$ uname
foo
```

- kill
